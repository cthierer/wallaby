1. create a bookmarklet by dragging the button from the home page to your browser's bookmark bar.

  2. on this page, click on the bookmarklet to load the application. this is similar to how the application would be loaded in the comic reader.

    a new button should be added to the toolbar below.

  3. if the icon is a key, then you must authenticate - click the key to open a new window and log into GitHub.

    the application only keeps your GitHub username and user ID.

  4. once logged in, a bookmark icon should appear in the toolbar - clicking it will save a new bookmark.

    the sample application includes hardcoded page and metadata values for testing only - other functionality does not work.

  5. return to the home page and check that the new bookmark was added to the "recent bookmarks" list.
